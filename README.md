# Lern Deutsch mit Arash — Aspekte Beruf B2

Begleitende Online-Übungen zum Lehrbuch **Aspekte Beruf B2** (Klett).
Erstellt von Arash Guitoo · veröffentlicht über GitHub Pages.

## Inhalt

- `index.html` — Startseite mit Buchcover und Übersichtskarten
- `cover.jpg` — Buchcover
- `lektion1-uebung.html` — **Lektion 1: Mein Beruf** (4 Module, 120 Aufgaben)

## Lektion 1 — Mein Beruf

| Modul | Thema | Schwerpunkt |
|-------|-------|-------------|
| 1 | Pendeln oder umziehen? | Wortstellung / Mittelfeld, Meinung äußern |
| 2 | Bei der Agentur für Arbeit | Telefongespräch, Behördenbriefe, sich vorstellen |
| 3 | Das kann ich gut | Soft Skills, Stellenanzeigen, Negation |
| 4 | Mein Abschluss | Anerkennung, Anfrage-E-Mail ordnen |

**Funktionen**
- 4 Module à 30 Aufgaben, durch Passwörter nacheinander freigeschaltet
- Aufgabentypen: Lesetext, Multiple Choice, Lückentext, Zuordnen, Wortsalat, Reihenfolge (E-Mail/Gespräch ordnen)
- Endless-Stack: falsch beantwortete Aufgaben kommen am Modulende erneut
- Fortschritt wird lokal gespeichert (localStorage)
- Namenseingabe und herunterladbares Zertifikat **nach jedem Modul** sowie ein Abschluss-Zertifikat
- Mobil-optimiert (Smartphone-tauglich)

## Veröffentlichen über GitHub Pages

1. Diese Dateien in das Repository hochladen (Wurzelverzeichnis oder Ordner `docs/`).
2. Im Repository unter **Settings → Pages** als Quelle den passenden Branch/Ordner wählen.
3. Die Seite ist anschließend unter `https://<benutzername>.github.io/<repo>/` erreichbar.
